# Results Summary

The validated run assigned POIs to 5,601,823 of 5,610,316 activity rows, a
99.84% assignment success rate. It covers 1,402,579 synthetic people and all
14 activity types represented in the final workflow.

The assigned set uses 49,157 distinct POIs. The simulation export lookup
contains 55,717 POIs because it also includes reference locations needed by
the downstream simulation workflow.

Median non-home travel distance is 0.853 km and 76.7% of non-home activities
occur in a different TAZ from home. Mean confidence is 4.79 out of 5, with
96.3% of rows receiving a confidence score of 4 or 5.

See [../validation/validation_report.md](../validation/validation_report.md)
and [../visualizations/README.md](../visualizations/README.md) for supporting
evidence.
