## Interpretation

The relative trajectory/TAZ correction preserves genuine cross-zone travel
while avoiding direct transfer of absolute destination zones from real agents
who live elsewhere. Activity-specific distance patterns remain plausible, and
the hierarchical POI fallback maintains high coverage.

The full result and input mobility data are not committed because of GitHub
file limits and source-data licensing. Included JSON, text reports, samples,
and figures provide auditable public evidence.
