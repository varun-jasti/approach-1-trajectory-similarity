# Validation

Validation combines assignment coverage, travel-distance checks, TAZ
separation, POI diversity, confidence scoring, and distribution comparisons
against Veraset observations.

- `ks_distribution_tests.json`: statistical distribution tests
- `validation_report.md`: concise interpretation of validated results
- `similarity_matching_report.txt`: production similarity-matching report
- `taz_fix_validation.txt`: relative-offset correction validation
- `poi_diversity_validation.txt`: assigned POI diversity analysis
- `confidence_metrics.txt`, `distance_metrics.txt`: detailed quality summaries
- `sample_validation.py`: validates the included public CSV sample
